### АСУУЛТ
1. BACKUP-ийн ач холбогдол
- Backup бол  "Нөөц дугуй" юм.
- Машин явж байхад дугуй хагарах эрсдэлтэй байдаг шиг, компьютер дээр ажиллахад файл устах эрсдэл үргэлж байдаг.
- Өгөгдөл алдагдахаас хамгаалах (Data Loss Prevention)
Хамгийн энгийн бөгөөд чухал шалтгаан. Мэдээлэл дараах шалтгаанаар устах эрсдэлтэй:

- Техник хангамжийн эвдрэл: Hard drive эвдрэх, компьютер шатах.

- Хүний хүчин зүйл: Санамсаргүй байдлаар файл устгах, форматын алдаа гаргах.

- Хулгай болон байгалийн гамшиг: Төхөөрөмжөө алдах, гал түймэр, усны аюул.

### ЖИШЭЭ
- Хагас жил бэлдсэн дипломын ажлаа ганцхан флэш дээр хадгалж байтал флэш нь эвдэрчихэв. Хэрэв тэр файлаа Google Drive руу хуулсан байсан бол 5 минутын дотор буцаагаад авчихна.
